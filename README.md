# Spring-Core-Practice

A basic Spring Core project demonstrating key concepts like Inversion of Control (IoC) and Dependency Injection (DI) using XML and Java-based configuration.

---

## 🚀 Features
- Understanding of Spring Core fundamentals
- Implementation of IoC (Inversion of Control)
- Dependency Injection (DI)
- Bean configuration using XML
- Java-based configuration (if included)
- Bean lifecycle basics

---

## 🛠️ Technologies Used
- Java
- Spring Core
- Maven
- Eclipse / IntelliJ IDEA
